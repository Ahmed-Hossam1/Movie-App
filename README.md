# Movieo App

Welcome to **Movieo App**, a modern movie and TV series explorer built with cutting-edge web technologies. Discover the latest movies, explore TV shows, search for your favorite titles, read reviews, and manage your account seamlessly. This application offers a rich, fully responsive user experience across all devices.

## 🚀 Features

- **Explore Movies & TV Series**: Browse through a vast collection of movies and TV shows.
- **Detailed Information**: View comprehensive details about any movie, TV series, or person (actor/crew).
- **Search Functionality**: Quickly find specific movies or TV shows using the integrated search.
- **User Authentication**: Securely log in or create a new account to unlock features.
- **Responsive Design**: Enjoy a flawless, mobile-friendly experience on any screen size.
- **Interactive UI**: Featuring smooth sliders, alerts, and modern components for an engaging user experience.

## 🛠️ Tech Stack & Key Dependencies

This project is built using powerful modern tools. Below are the key packages and their official documentation:

*   **[React](https://react.dev/)**: The core library for building our dynamic user interface.
*   **[React Router DOM](https://reactrouter.com/)**: Enables smooth, client-side routing between pages (Home, Search, Details, etc.).
*   **[Tailwind CSS](https://tailwindcss.com/)**: A utility-first CSS framework used for rapid, responsive styling.
*   **[DaisyUI](https://daisyui.com/)**: A component library for Tailwind CSS that provides clean, pre-built component classes.
*   **[Axios](https://axios-http.com/docs/intro)**: A promise-based HTTP client for making API requests to fetch movie data.
*   **[Swiper](https://swiperjs.com/)**: A modern mobile touch slider with hardware-accelerated transitions.
*   **[SweetAlert2](https://sweetalert2.github.io/)**: Used for displaying beautiful, responsive, and customizable popups/alerts.

## 📦 Installation & Setup

Want to run Movieo App locally? Follow these simple steps:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/Ahmed-Hossam1/Movie-App.git
    ```
2.  **Navigate into the project directory:**
    ```bash
    cd Movie-App
    ```
3.  **Install the necessary dependencies:**
    ```bash
    npm install
    ```
4.  **Set up your Environment Variables:**
    Create a `.env` file in the root directory and add securely configure your API credentials.

5.  **Start the development server:**
    ```bash
    npm start
    ```
6.  Open [http://localhost:3000](http://localhost:3000) in your browser to view the app!
